        ✅ 1. let a = 10;
This just sets the value of a to 10 directly — it's hardcoded in your program. It doesn't ask the user anything.

✅ 2. let a = prompt("Enter 1st number");
This asks the user to type a value, like this:

A popup appears in the browser that says "Enter 1st number"

Whatever the user types gets saved in the variable a

🟡 Important Note:
prompt() always returns a string — even if the user types a number.

So if we want to do math, we should convert it to a number:

javascript
Copy
Edit
let a = Number(prompt("Enter 1st number"));
let b = Number(prompt("Enter 2nd number"));

let sum = a + b;
console.log("Sum is:", sum);
